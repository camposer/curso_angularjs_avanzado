Cómo configurar el proyecto
===========================

0.- Crear carpeta donde se clonará el repositorio

1.- Clonar repositorio (URL del repositorio)
> git clone https://github.com/camposer/curso_angularjs_avanzado.git

2.- Iniciar backend
> cd ejercicio\backend
> npm install
> node app

Abrir en un navegador:
	http://localhost:3000/productos

3.- Iniciar frontend (sugerencia: abra otra consola)
> cd ejercicio\frontend
> npm install
> npm start

Abrir en un navegador:
	http://localhost:8000/app
